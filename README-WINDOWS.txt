mario_task -- Windows install
============================

1.  Double-click  install.bat
    (If Windows SmartScreen warns about an unknown publisher, click
     "More info" then "Run anyway".)

2.  Wait for "Setup complete!"   (~5 minutes; downloads ~1.5 GB total)

3.  Double-click  "Run Mario Task"  on your desktop.
    First launch will open the configuration wizard.

That's it. For troubleshooting, see README.md (also in this folder).
